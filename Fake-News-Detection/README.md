# Fake News Detection using Machine Learning

## Project Overview
This project builds a machine learning model that can classify news articles as **Fake** or **Real** using Natural Language Processing (NLP) techniques.

The model is trained on a dataset of real and fake news articles and uses **TF-IDF vectorization** along with **Logistic Regression** for classification.

---

## Dataset

The dataset used for this project contains two files:

Fake.csv – contains fake news articles  
True.csv – contains real news articles

Each dataset includes:
- Title
- Text
- Subject
- Date

Source: Kaggle Fake and Real News Dataset

---

## Technologies Used

- Natural Language Processing (NLP)

1. Data Collection

---

## Model Performance

The model achieves approximately **90–95% accuracy** in classifying fake and real news.

Evaluation metrics used:
- Accuracy Score
- Confusion Matrix
- Classification Report
